# คู่มือเริ่มต้น Accounting Automation

การทำ Automation ไม่จำเป็นต้องเริ่มจากระบบใหญ่ เริ่มจากจุดเล็กๆ ที่สร้างผลลัพธ์ได้เร็ว

## ขั้นตอนการเริ่มต้น

1. สำรวจกระบวนการที่ซ้ำซ้อน
2. เลือกเครื่องมือที่เหมาะสม
3. เริ่มจากกระบวนการเดียว
4. วัดผลและขยายต่อ
